# ft_printf

Reimplementation of `printf()` supporting the core format specifiers using variadic functions.

---

## Components

| Note | Description |
|---|---|
| [[Projects/ft_printf/Entry Point\|Entry Point]] | `ft_printf()` — parses format string, dispatches to handlers, returns char count |
| [[Projects/ft_printf/Helpers\|Helpers]] | `print_handler()`, `print_chr()`, `print_nbr()`, `ft_putnbr_base()` |

---

## Supported Specifiers

`%c` `%s` `%p` `%d` `%i` `%u` `%x` `%X` `%%`
